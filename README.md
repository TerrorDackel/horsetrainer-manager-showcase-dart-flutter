# HorseTrainer Manager (Showcase)
Professional stable and horse training operations platform currently in active development.
## Status
- Stage: Private development and staging validation
- Availability: Not publicly released
- Repository type: Showcase only (no production source code)
## Vision
HorseTrainer Manager is built to support professional trainers and stables with:
- horse records and lifecycle workflows
- customer and contract management
- calendar and operational planning
- office and billing support
- role-based access and legal/compliance safeguards
## Current Focus
- UX consistency across mobile and desktop
- reliable navigation and access control
- legal workflow hardening (consent and document handling)
- high localization quality (German/English)
## Tech Snapshot
- Flutter (Web + Mobile)
- Firebase (Auth, Firestore, Hosting, Storage)
- CI/CD via GitHub Actions
- structured sprint documentation and staged rollout process
## Why this repository is public
This repository is a **public showcase**.
It contains:
- roadmap progress
- product updates
- selected screenshots/visual snapshots
It does not contain:
- production source code
- internal infrastructure details
- proprietary business logic
